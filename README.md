Global Startup and Unicorn Analysis:

Overview

This project analyzes global startup data to understand how companies become unicorns — startups valued at over $1 billion.
The goal is to explore valuation trends, funding patterns, geographic distribution, and how long startups take to reach unicorn status.
The project uses SQL for data cleaning and analysis, and Power BI for dashboard visualization.

Objectives

Identify which countries and industries produce the most unicorns.
Analyze average valuation and funding levels across industries.
Calculate the average years it takes startups to reach unicorn status.
Visualize global trends in unicorn creation over time.
Build an interactive dashboard to summarize insights.

Tools and Technologies

Python (pandas, kagglehub) – Downloading and preparing the dataset.
SQLite – Data storage and SQL-based cleaning.
SQL – Data wrangling and analysis.
Power BI – Data visualization and dashboards.
DB Browser / VS Code – SQL script execution and database management.

Data Source

Dataset: Startup Growth and Investment Data from Kaggle
File used: Unicorn_Companies.csv
